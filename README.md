# Spam Mail Detection using SVM

A machine learning web application that detects spam emails using a Support Vector Machine (SVM) classifier.

## Features
- Email spam classification
- TF-IDF feature extraction
- SVM decision boundary visualization
- Confidence score prediction
- Flask web interface

## Tech Stack
Python
Scikit-learn
Flask
HTML/CSS
Matplotlib

## Model
Algorithm: Support Vector Machine (LinearSVC)
Feature Extraction: TF-IDF

## Run Locally

pip install -r requirements.txt
python app/app.py
