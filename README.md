#  Spam Mail Detection using SVM

A machine learning web application that detects whether an email message is **Spam or Not Spam** using a **Support Vector Machine (SVM)** classifier.

The model is trained using TF-IDF text features and deployed as a **Flask web application**.

---

##  Live Demo

Try the deployed application here:

🔗 https://spam-mail-detection-svm.onrender.com

(Note: The free Render instance may take ~30–50 seconds to wake up if inactive.)

---

##  Features

- Spam email detection using **Support Vector Machine**
- **TF-IDF feature extraction**
- **Confidence score prediction**
- **Interactive web interface**
- **SVM decision boundary visualization**
- Hosted online using **Render**

---

##  Machine Learning Model

Algorithm used:

Support Vector Machine (LinearSVC)

Feature Engineering:

TF-IDF Vectorization

Evaluation Metric:

Accuracy: 1.0 (on test dataset)
